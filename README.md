<h1 align="center">BEUTIFULFILM</h1>

Интерактивная страница, для поиска фильмов, с возможностью добаления в избранное и фильтрацией. Реализован функционал регистрации, авторизации, редактирования профиля. 
  
Ссылка на сайт: [https://zaigraev.movie.nomoredomains.work](https://zaigraev.movie.nomoredomains.work)

<h3 align="center">Технологии</h3>

- HTML5 - cемантическая вёрстка
- SASS (Flexbox) - Наименования элементов и структура по методологии БЭМ Nested. Анимации элементов. Интерфейс адаптирован под различные устройства ( от 320 до 1280 и более пикселей по ширине).
- JavaScript ES6 - функциональный подход, async/await, fetch API.
- React - функциональные компоненты.
  - Функции Context, Ref, Redirect, Route, Switch.
  - Хуки useState, useEffect, useContext, useRef, useCallback, useNavigate.
  - Састомные хуки useInput, useValidation
  - Авторизация, валидация, защита роутов.
  - LocalStorage - хранения данных между сессиями.

Бекенд проекта: [https://github.com/VladimirZaigraev/movies-explorer-api](https://github.com/VladimirZaigraev/movies-explorer-api)
